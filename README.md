## 🏥 项目一：智慧养老服务平台（团队项目）

### 📌 项目简介
面向社区老人的综合养老服务系统，提供护理服务预约、房间分配、服务查询等功能，后端 API 供微信小程序调用，实现服务全流程可视化。

### 🔧 核心功能
- 用户注册/登录、护理员管理、服务类型配置  
- **智能预约机制**：防止同一房间/护理员被重复预约  
- 实时查询可用房间与服务状态  
- 订单状态同步至小程序前端，支持取消与修改  

### ⚙️ 技术实现
- **后端框架**：Spring Boot + Spring MVC + MyBatis-Plus  
- **数据库**：MySQL（设计用户、护理员、订单、房间等核心表）  
- **缓存优化**：Redis 缓存热门服务与房间库存，接口响应时间降低 60%  
- **并发控制**：基于数据库行锁 + 事务隔离级别，解决超订问题  
- **前端对接**：提供 RESTful API，供微信小程序调用  

### 💡 AI 延展思考
> 当前正探索基于用户历史行为（如预约频率、服务偏好、健康数据），利用大模型（如 Qwen）生成个性化护理建议与智能排期方案，提升服务精准度。
